Prerequisites

Before we dive into coding, you need to set up your environment. Web development is accessible to everyone, but having the right tools makes it easier.

1. Install a Code Editor

You need a text editor to write your code. We recommend VS Code because it is free, powerful, and has great extensions for HTML and CSS.

2. Web Browser

You need a browser to view your website. Google Chrome or Firefox are best for developers because of their powerful "Inspect Element" tools.
